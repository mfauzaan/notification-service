## Installation

- Set .env variables based on .env.example
- Make sure that payment service is reachable and correct port & host is set on the env
- For swagger docs `/docs`

```bash
$ npm install
$ npm run start:dev
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
